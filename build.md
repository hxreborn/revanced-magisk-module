## Installation

### Non-root APKs
1. Install [MicroG](https://github.com/ReVanced/GmsCore/releases) (required for YouTube/Music)
2. Install the APK file
3. Grant all permissions when prompted

### Magisk Modules
1. Flash the  file in Magisk Manager
2. Reboot device
3. Install [zygisk-detach](https://github.com/j-hc/zygisk-detach) to prevent Play Store updates

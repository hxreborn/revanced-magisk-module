<summary><strong>Installation</strong></summary>

**Non-Root (APK)**
1. Install [MicroG](https://github.com/ReVanced/GmsCore/releases) (YouTube/Music only)
2. Install APK

**Root (Magisk)**
1. Flash `.zip` in Magisk
2. Reboot
3. Use [zygisk-detach](https://github.com/j-hc/zygisk-detach) to prevent Play Store updates
